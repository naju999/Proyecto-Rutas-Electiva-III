# React Shell Paralelo

Este proyecto es el cascaron React en paralelo para la fase 1 de migracion.

## Objetivo

- Mantener el frontend actual sin tocar.
- Levantar navegacion React con rutas base.
- Dejar pantallas estructuradas sin logica de negocio aun.

## Comandos

```bash
npm install
npm run dev
```

## Rutas

- /inicio
- /rutas
- /favoritos
- /perfil
